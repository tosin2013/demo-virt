= Virtual Machine Templates

== Uploading a VM disk image

https://docs.redhat.com/en/documentation/openshift_container_platform/4.18/html/virtualization/advanced-vm-creation#virt-uploading-image-virtctl_virt-creating-vms-uploading-images[Docs]

[source,bash]
----
virtctl image-upload dv <datavolume_name> \
  --size=<datavolume_size> \
  --image-path=</path/to/image> \
  --insecure
----

== Create a custom disk image for VM

https://docs.redhat.com/en/documentation/openshift_container_platform/4.18/html/virtualization/advanced-vm-creation#virt-creating-vms-uploading-images[Docs]

Generalizing a boot image
```bash
VM_NAME=rhel9-model
virtctl stop $VM_NAME
oc get vm $VM_NAME -o jsonpath="{.spec.template.spec.volumes}{'\n'}"
oc get pvc
virtctl guestfs <my-vm-volume> --uid 107
```
